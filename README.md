Hi if you wanna use this it's in a super mega testing phase rn, but run " python3 -m midi_editor.app " in the app folder and it will launch.
controls are scroll wheel to scroll view, CMD +scroll to zoom x axis and ALT +scroll to
zoom the other axis. just have import a midi file mess around and have some fun this is not a real build.

Install it like this:
git clone https://github.com/Hackerman696/Porymusic
cd <repo-folder>
python3 -m venv .venv
source .venv/bin/activate
python -m pip install -U pip
pip install -r requirements.txt

run it like this from the root:
python -m midi_editor.app
